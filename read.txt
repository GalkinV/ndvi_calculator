Для работы модуля нужно:

1. Установить гдал и гдал-дев
sudo apt install gdal-bin
sudo apt install gdal-dev

2. Настроить виртуальное окружение
pip install virtualenv
virtualenv venv
source venv/bin/activate

3. Установить Numpy
pip install Numpy

4. Установить гдал, предварительно узнав нужную версию через ogrifo
export C_INCLUDE_PATH=/usr/include/gdal
export CPLUS_INCLUDE_PATH=/usr/include/gdal
ogrinfo --version
pip install GDAL==XXX

5. Установить pyModis
pip install pyModis

6. Тест обработает 2 файла из директории HGT и туда же создаст tiff'ы
python3 run.py
